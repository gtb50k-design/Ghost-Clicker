# 👻 Ghost Clicker

Ghost Clicker is a lightweight auto clicker designed to keep things simple, fast, and out of your way. Whether you're gaming, testing, or just dealing with repetitive clicking, it gets the job done without extra clutter.

## ✨ Features
- Adjustable click speed (CPS)
- Easy start/stop hotkey
- Runs in the background
- Lightweight and low CPU usage
- Simple, clean interface

## 🚀 How to Use
1. Open Ghost Clicker
2. Set your desired click speed
3. Press the hotkey to start/stop clicking
4. Let it run while you focus on other tasks

## 📌 Notes
- Use responsibly

## ⚡ Purpose
Made to save time on repetitive clicking and keep things smooth and simple.
